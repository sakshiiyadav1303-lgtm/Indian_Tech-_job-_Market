# 📊 Indian Tech Job Market Analysis 2026

An interactive **Data Analytics Dashboard** built using **Python, Pandas, Plotly, and Streamlit** to analyze hiring trends, salary patterns, skills demand, company insights, and market trends in the Indian technology sector.

---

# 📌 Project Overview

The **Indian Tech Job Market Analysis 2026** project aims to explore and visualize technology job listings across India through an interactive dashboard.

The project includes:

- Data Cleaning
- Exploratory Data Analysis (EDA)
- Interactive Streamlit Dashboard
- Project Report
- Project Presentation

Users can explore job market trends using interactive filters and visualizations.

---

# 🎯 Project Objectives

- Analyze salary trends in the Indian technology industry.
- Identify the most in-demand technical skills.
- Explore hiring trends across major Indian cities.
- Analyze company hiring patterns and ratings.
- Understand work mode distribution (Remote, Hybrid, On-site).
- Build an interactive dashboard for job market exploration.

---

# 🚀 Installation

## Clone the repository

```bash
git clone https://github.com/yourusername/Indian-Tech-Job-Market-Analysis-2026.git
```

---

## Navigate to the project folder

```bash
cd Streamlit
```

---

## Install required packages

```bash
pip install -r requirements.txt
```

---

## Run the Streamlit Dashboard

```bash
streamlit run Streamlit/Indian_Tech_Job_Market_Dashboard.py
```

# 💡 Key Insights

- Mumbai is one of the leading technology hiring hubs.
- Python remains one of the most demanded programming skills.
- Mid-level experience candidates receive the highest number of opportunities.
- Larger companies generally have higher hiring volumes.
- Salary varies significantly based on experience and company size.
